# Quiz App - Android Java

A dynamic Quiz App built with Android Studio and Java. 
Tests users on Networking and Software Development concepts.

## Features
- User Registration
- Specialization Selection (Network / Software)
- 10 MCQ Questions per category
- Countdown Timer (10 minutes)
- Real-time Score Tracking
- Results Page with Percentage & Time Taken
- Satisfaction Rating with SeekBar

## Tech Stack
- Language: Java
- IDE: Android Studio
- Components: RadioButtons, SeekBar, CountDownTimer, Intents
